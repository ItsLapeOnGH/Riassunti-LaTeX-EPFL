# STEM University Notes (LaTeX)

This repository contains a collection of concise summaries of STEM subjects written in LaTeX.
The goal is to provide clear and structured notes for university-level courses.

## Contents

The notes may include topics from areas such as:

* Mathematics
* Physics
* Computer Science
* Other STEM-related subjects

## Structure

The project is written entirely in LaTeX and organized to keep the source files clean and modular.

## Purpose

These notes are intended as a quick reference and study aid for reviewing key concepts and formulas.

## Compilation

To compile the document locally:

```
pdflatex main.tex
```
